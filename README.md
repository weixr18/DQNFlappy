# DQNFlappy
A Deep Q Network Learning how to Play Flappy Birds

## 文件结构
+ train_torch.py 主要代码文件
+ assets 游戏所需组件
+ game 游戏代码
+ tensorboard 网络结构log文件

## Packages
+ tensorflow==1.x
+ opencv
+ numpy
+ pygame

## 准备工作
+ 将./main.py文件中的 SP 变量改为自己电脑上的项目文件夹位置
+ 将./game/flappy_birds_utils.py文件中的 SP 变量改为自己电脑上的项目文件夹位置

## 运行
  python main.py
